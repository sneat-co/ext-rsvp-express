# RSVP Express contract

This repository publishes the public TypeScript/Angular contract for the RSVP
Express Sneat extension: DTOs, navigation contexts, service interfaces, and
injection tokens. Runtime services and UI remain private in
[`sneat-co/rsvp-express`](https://github.com/sneat-co/rsvp-express).

## Package

Install [`@sneat/extension-rsvpexpress-contract`](https://www.npmjs.com/package/@sneat/extension-rsvpexpress-contract)
to depend on RSVP Express without taking a dependency on its runtime
implementation.
